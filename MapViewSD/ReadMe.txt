MapViewSD Installation Instructions

1) Unzip the archive into a folder
2) Download and install MS Visual C++ Redistributable - https://aka.ms/vs/17/release/vc_redist.x64.exe (if you don't have Visual Studio 2022 installed)
3) Run ODBC Data Sources (64-bit) and create a Data Source called "TigerData". Point it to the Tiger.accdb file in your folder
4) Run MapViewSD and open up the sample Knox-Waldo2006Maine.gdb file
5) Follow the demo on the website
